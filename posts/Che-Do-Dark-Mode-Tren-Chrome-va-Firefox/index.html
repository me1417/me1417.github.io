    [Hướng dẫn] Cách Sử Dụng Chế Độ Dark Mode Trên Chrome Và Firefox

    Dark Mode là chức năng chuyển đổi tông màu, biến mọi trang web thành nền tối dù trang web đó có hỗ trợ hay không. Dark Mode thường dùng vào ban đêm hoặc điều kiện thiếu sáng giúp dịu mắt hơn hoặc đôi khi chỉ là để thay đổi giao diện cho đỡ nhàm chán.

    Hướng dẫn sử dụng chế độ Dark Mode trên Chrome và Firefox

    Trên Chrome

    Cách 1:

    Từ bản Chrome 78 trở đi bạn có thể kích hoạt chế độ Dark Mode mặc định, bạn vào chỉnh setting của Chrome bằng cách nhập đường dẫn chrome://flags/#enable-force-dark. Sau đó kích hoạt chức năng này lên.

    Tuy nhiên hiện nay mình chưa thấy phím tắt để bật tắt nhanh chế độ này do đó mình vẫn dùng theo cách 2, vì tính năng này vẫn chưa được hoàn thiện khi chạy trên một số website.

    Cách 2:

    Dùng Chrome Extension. Cách này không liên quan đến chế độ mặc định của Chrome.

    Các bạn tìm extension Dark Reader trên Chrome Web Store hoặc vào link này.

    Cách này thì bạn có thể dùng phím tắt và tùy chỉnh phím tắt. Hoặc bật tắt chế độ riêng cho từng trang web.

    Trên Firefox

    Các bạn tìm add-on Dark Reader trên Addons Mozilla hoặc vào link này.
